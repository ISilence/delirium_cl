## Synopsis

Memory managment library for heterogeneous architectures using OpenCl 

## Requirements

C++11 compiler
OpenCl SDK + driver

## Installation

```
cd <project_dir>
mkdir build
cd ./build
cmake ../
make 
cd ../
# ./bin/benchmark # run example
```
